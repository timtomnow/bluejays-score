Blue Jays Score puts the Blue Jays' most relevant game first, then the rest of the league around it, on a single scrolling screen. This tour shows you what each part is.

## Steps

1. Open the app. The top **game card** shows the Blue Jays' most relevant game — a live game if one is on, otherwise the next scheduled game or the latest final. On a day with no game it reads **No game today. Rest up, Blue Jays.**
2. Read the **badge** at the top of the card — **Live**, **Final**, or the scheduled start — to know the game's state.
3. Scroll down to the **Season** card for the Blue Jays' record, current streak, and the **AL East** standings.
4. Below that, find **Last 5 Games** and **Upcoming Games**.
5. Keep scrolling for **Other MLB Games Today** — every other game around the league.
6. Tap **Refresh** at the bottom any time to reload immediately.

## What you'll see

A full snapshot of the Blue Jays' current game plus the league around them. The app refreshes on its own — every 30 seconds while a game is live, every minute around a scheduled start, and every 5 minutes otherwise — so you rarely need to refresh by hand.
